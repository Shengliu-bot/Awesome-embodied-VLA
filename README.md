* **RoboNurse-VLA**: "RoboNurse-VLA: Robotic Scrub Nurse System based on Vision-Language-Action", *arXiv, Sep 2024*. [[Paper](https://arxiv.org/abs/2409.19590) ][[Website](https://arxiv.org/abs/2409.19590)]
